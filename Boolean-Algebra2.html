Boolean Algebra – A Level Computer Science (9618)

1. Introduction

Boolean Algebra is the mathematical system used to analyse and simplify logic circuits. It deals only with two values: 0 (False) and 1 (True). Every digital circuit—adders, flip-flops, CPUs, memory—uses Boolean operations internally.

This page provides detailed explanations, teacher notes, student-friendly steps, examples, activities, and exam-style questions.

2. Basic Boolean Operations

2.1 AND Operation

Output is 1 only when all inputs are 1.

2.2 OR Operation

Output is 1 if any input is 1.

2.3 NOT Operation

Inverts the input: 0→1 and 1→0.

3. Boolean Algebra Laws

3.1 Identity Laws

A + 0 = A

A · 1 = A

3.2 Null Laws

A + 1 = 1

A · 0 = 0

3.3 Idempotent Laws

A + A = A

A · A = A

3.4 Complement Laws

A + A' = 1

A · A' = 0

3.5 Commutative Laws

A + B = B + A

A · B = B · A

3.6 Associative Laws

A + (B + C) = (A + B) + C

A(B C) = (AB)C

3.7 Distributive Laws

A(B + C) = AB + AC

A + BC = (A + B)(A + C)

3.8 De Morgan’s Laws

(A B)' = A' + B'

(A + B)' = A' B'

4. Step-by-Step Simplification Examples

Example 1: Simplify A + A B

  1. Use Distributive Law: A + AB = A(1 + B)
  2. Use Identity Law: 1 + B = 1
  3. Final result: A

Example 2: Simplify (A + B)(A + B')

  1. Expand: A(A + B') + B(A + B')
  2. Distribute: AA + AB' + AB + BB'
  3. Simplify: A + AB' + AB + 0
  4. Factor A: A(1 + B' + B)
  5. 1 + anything = 1
  6. Final result: A

5. Teacher Notes

  • Always begin with truth tables before laws.
  • Provide students with circuit diagrams and ask them to convert to Boolean expressions.
  • Encourage identifying patterns—most A Level questions use De Morgan or Distributive laws.
  • Use colour-highlighting when showing simplification steps.

6. Student Activities

Activity 1 – Truth Table Construction

Build truth tables for:

  • A'B + AB'
  • (A + B)' + AB

Activity 2 – Expression to Circuit

Draw logic circuits for:

  • F = A(B + C')
  • F = (A + B')(C + A')

Activity 3 – Simplification

Simplify step-by-step:

  • A'B + AB + AB'
  • (A + B)(A' + C)

7. Exam-Style Questions (Scenario-Based)

Q1 (5 marks)

A traffic light controller uses sensors A (vehicle present) and B (pedestrian button pressed). The system outputs 1 for "allow pedestrian crossing". The logic is:

F = A'B + AB'

Explain the scenario in your own words and simplify the expression to its simplest form.

Q2 (5 marks)

A security system activates an alarm using doors A and B. The alarm should activate when exactly one door is open. The logic is:

F = A B' + A' B

Draw the logic circuit and provide the truth table.

Q3 (4 marks)

Given the expression used in a heating controller:

F = (A + B)' + AB

Simplify and state the output conditions.

Q4 (4 marks)

A manufacturing robot activates when:

  • Safety sensor A is ON
  • Obstacle sensor B is OFF
  • Manual override C is ON

Write the Boolean expression and simplify it.
