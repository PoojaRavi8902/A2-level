Short Answer Questions

1. Explain the difference between == and .equals() in Java.

Answer: == compares references (memory addresses), while .equals() compares object content when overridden properly.

2. What is the default value of an uninitialized boolean variable in Java?

Answer: false

3. What does the static keyword mean for a method?

Answer: It belongs to the class, not instances; can be called without creating an object.

4. What is the purpose of the finally block?

Answer: It always executes after try-catch, used for cleanup like closing resources.

5. What is method overriding?

Answer: A subclass provides a new implementation for a method defined in its superclass.

6. State one difference between an abstract class and an interface.

Answer: Abstract class can have implemented methods; interface methods are abstract by default (until Java 8).

7. What is a constructor in Java?

Answer: A special method with the same name as the class used to initialize objects.

8. What happens if you don’t provide a constructor in a class?

Answer: The compiler provides a default no-argument constructor.

9. What is polymorphism?

Answer: The ability of an object to take many forms, e.g., method overriding and overloading.

10. What is the purpose of the this keyword?

Answer: Refers to the current object instance.

11. What is garbage collection in Java?

Answer: Automatic process of reclaiming memory by destroying unused objects.

12. What is the difference between ArrayList and array?

Answer: Arrays are fixed size; ArrayLists are resizable and part of Java Collections.

13. Which keyword is used to create a subclass?

Answer: extends

14. What is the difference between checked and unchecked exceptions?

Answer: Checked exceptions are checked at compile time, unchecked occur at runtime.

15. What is encapsulation?

Answer: Bundling data and methods together, restricting direct access to fields.

16. What is the difference between String and StringBuilder?

Answer: String is immutable, StringBuilder is mutable and faster for modifications.

17. What does public static void main mean?

Answer: Entry point of a Java program; public = accessible, static = no object needed, void = no return, main = method name.

18. What is inheritance?

Answer: Mechanism where one class acquires properties and methods of another.

19. Which class is the superclass of all Java classes?

Answer: Object

20. What is a package in Java?

Answer: A collection of related classes and interfaces grouped together.

Exam-Style Questions

1. Write a Java program that reads 10 integers into an array and prints the largest number.

int[] arr = new int[10];
Scanner sc = new Scanner(System.in);
for (int i = 0; i < 10; i++) {
    arr[i] = sc.nextInt();
}
int max = arr[0];
for (int num : arr) {
    if (num > max) max = num;
}
System.out.println("Largest: " + max);

2. Explain the concept of method overloading in Java with an example.

Method overloading = multiple methods in the same class with the same name but different parameter lists.

Example:
class MathUtil {
    int add(int a, int b) { return a+b; }
    double add(double a, double b) { return a+b; }
}

3. Write a Java class BankAccount with fields accountNumber and balance. Add deposit() and withdraw() methods. Show usage in main().

class BankAccount {
    int accountNumber;
    double balance;

    BankAccount(int acc, double bal) {
        accountNumber = acc;
        balance = bal;
    }

    void deposit(double amt) { balance += amt; }
    void withdraw(double amt) { balance -= amt; }
}

public class Main {
    public static void main(String[] args) {
        BankAccount b = new BankAccount(123, 500);
        b.deposit(200);
        b.withdraw(100);
        System.out.println(b.balance);
    }
}

4. Describe the differences between abstract classes and interfaces in Java. Give an example use case for each.

Abstract class: can have concrete + abstract methods, use when classes share common behavior.
Interface: only method signatures (until Java 8 added default methods), use when unrelated classes share capability.

Example:
abstract class Animal { abstract void makeSound(); }
interface Flyable { void fly(); }

5. Write a Java program using try-catch-finally to handle division by zero exception.

import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        try {
            int a = sc.nextInt();
            int b = sc.nextInt();
            int c = a / b;
            System.out.println("Result: " + c);
        } catch (ArithmeticException e) {
            System.out.println("Division by zero not allowed!");
        } finally {
            System.out.println("Execution completed.");
        }
    }
}
