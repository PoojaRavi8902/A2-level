Chapter 18 — Artificial Intelligence (AI)

Teacher-friendly 60-minute lesson plan | Starter → Lecture → Activity → Lecture → Plenary
Estimated Class Time
60 minutes (15 | 10 | 15 | 12 | 8)

Phase 1 — Starter (12-15 minutes)

Purpose: To activate prior knowledge of decision-making systems, logic gates, and data structures from earlier chapters.

  1. Ask students: “What makes a system intelligent?” and collect 3-5 answers on the board.
  2. Play a 5-minute quick quiz (use Mentimeter, Kahoot, or paper) with questions like:
    • What does a truth table show?
    • How does a computer make a decision?
    • Give an example of an algorithm that can ‘learn’.
  3. Show two short clips (e.g., a robot vacuum and a chess-playing AI). Ask: “Which one truly ‘thinks’?”
Teacher notes:
  • Identify misconceptions — most students think AI means ‘robots’. Emphasise software intelligence.
  • Bridge to today’s goal: “We’ll explore what AI really means and how it makes decisions.”

Phase 2 — Teacher Lecture 1 (10 minutes)

18.1 Introduction to Artificial Intelligence

Artificial Intelligence (AI) refers to computer systems designed to perform tasks that normally require human intelligence. These include learning, reasoning, problem-solving, perception, and understanding language.

  • AI systems process large amounts of data and identify patterns.
  • They can make decisions and improve performance over time.
  • Examples: voice assistants, self-driving cars, spam filters, and recommendation systems.
Lecture tips:
  • Use real-life visuals (e.g., Siri, Google Translate, Netflix recommendations).
  • Ask students to name AI tools they already use daily.
  • Explain that AI ≠ automation. Automation follows fixed rules; AI adapts and learns.

18.2 Components of an AI System

  • Knowledge Base: facts, rules, and relationships.
  • Inference Engine: applies logic to deduce conclusions.
  • Learning Component: improves through experience or data.
  • User Interface: interacts with users and presents results.
Illustration idea:

Draw AI System Architecture diagram — arrows between KB → Inference → Learning → UI.

Phase 3 — Student Activity (15 minutes)

Interactive Task — Build a Mini Expert System (group work)

  1. Split class into small groups.
  2. Give each group a domain (e.g., plant disease, car maintenance, choosing a laptop).
  3. Ask them to identify 3 rules for decisions (IF… THEN…).
  4. Groups exchange their rule sets and test each other’s systems.
  • Encourage creativity — even humorous domains work well.
  • Use a whiteboard to show how simple expert systems mimic reasoning.
  • Students write the pseudocode of their rule-based decision system.

Phase 4 — Teacher Lecture 2 (12 minutes)

18.4 Machine Learning

Machine Learning (ML) allows computers to learn from data without explicit programming. It’s divided into three main categories:

  • Supervised Learning: trained on labelled data (input-output pairs). Example: spam detection.
  • Unsupervised Learning: finds patterns or clusters in unlabelled data. Example: market segmentation.
  • Reinforcement Learning: learns through trial and error with rewards and penalties. Example: AI in games like Chess or Go.
# Example (simplified supervised learning)
data = [(2,4),(3,6),(4,8)]  # input-output pairs
model = lambda x: x*2
print(model(5))  # predicts 10

18.5 Knowledge Representation

Knowledge representation is how AI stores and processes information:

  • Rules and Facts: IF temperature > 38 THEN fever.
  • Semantic Networks: Concept maps connecting ideas.
  • Frames: Data structures representing attributes of objects.
  • Ontologies: Hierarchical models showing relationships between concepts.
Extension example:

Demonstrate a mini semantic network on board — e.g., Animal → Mammal → Dog → Labrador.

18.6 Expert Systems

An expert system simulates human expertise using rules and logic.

  • Knowledge Base — domain information.
  • Inference Engine — decision logic.
  • User Interface — dialogue with user.

Example: A medical diagnosis system analyses symptoms and outputs likely diseases.
