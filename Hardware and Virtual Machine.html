18.02 Pipelining Example (RISC)

Pipelining allows multiple instructions to overlap in execution:

IF
Instruction Fetch
ID
Instruction Decode
OF
Operand Fetch
IE
Instruction Execute
WB
Write Back

Each stage operates simultaneously on different instructions, increasing throughput.
