Arrays / Lists in Python – Cambridge AS Level 9618

Teacher Pane – Concept & Explanation

1. What is an Array / List?

An array (called a list in Python) stores multiple values in a single variable.

Values are stored in order.

2. Key Characteristics (Exam Points)
  • Elements are in a fixed order
  • Each element has an index
  • Same data type (theory)
  • Fixed size in theory
  • Dynamic size in Python
In exams, say "fixed size". In Python, explain it can grow.
3. Creating a List in Python 
marks = [65, 70, 80, 90]

This list stores four student marks.

4. Accessing Elements using Index 
print(marks[0])   # First element
print(marks[2])   # Third element

Index always starts from 0.

5. Updating an Element 
marks[1] = 75

The second value is changed.

6. Why Arrays are Important in Exams
  • Used in most algorithms
  • Easy to trace
  • Foundation for other structures

Teacher Pane – Advanced & Exam-Oriented Use of Lists

7. Traversing a List (Core Exam Skill) 
marks = [65, 70, 80, 90]

for mark in marks:
    print(mark)
8. Index-Based Traversal 
for i in range(len(marks)):
    print(i, marks[i])
9. Searching for a Value 
target = 80
found = False

for mark in marks:
    if mark == target:
        found = True

print(found)
10. Finding Maximum 
maximum = marks[0]

for mark in marks:
    if mark > maximum:
        maximum = mark

print(maximum)
