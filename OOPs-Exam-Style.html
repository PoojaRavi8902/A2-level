Question 1(b) – Superclass Code [5]

Write program code for the class definition of the superclass BankAccount.

Expected Answer (Python): 
class BankAccount:
    def __init__(self, iban, name):
        self.__IBAN = iban
        self.__AccountHolderName = name

    def SetAccountHolderName(self, name):
        self.__AccountHolderName = name

    def GetAccountHolderName(self):
        return self.__AccountHolderName

    def GetIBAN(self):
        return self.__IBAN

Question 3(b) – NodeClass Definition [10]

Write program code to define NodeClass, including constructor, getters and setters.

Expected Answer (Python): 
class NodeClass:
    def __init__(self):
        self.__Data = ""
        self.__Pointer = -1

    def SetData(self, d):
        self.__Data = d

    def SetPointer(self, x):
        self.__Pointer = x

    def GetData(self):
        return self.__Data

    def GetPointer(self):
        return self.__Pointer
