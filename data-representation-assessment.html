Part A — Multiple Choice Questions (25)

  1. 1. How many distinct values can be stored in an 8-bit unsigned integer?

    Correct answer: B — 256 (values 0 to 255).

  2. 2. Which encoding is variable-length and backward compatible with ASCII?

    Correct answer: C — UTF-8.

  3. 3. Which colour depth most closely corresponds to “true color”?

    Correct answer: C — 24-bit (≈16.7 million colours).

  4. 4. Which of these is a lossless compression method?

    Correct answer: C — RLE (Run Length Encoding) is lossless.

  5. 5. What does the term ‘normalisation’ mean in floating-point representation?

    Correct answer: B — shift so mantissa uses full precision (MS fraction bit = 1).

  6. 6. Which file organisation allows direct access using hashing?

    Correct answer: C — Direct access (random access) can use hashing.

  7. 7. Which is TRUE about binary files vs text files?

    Correct answer: B — text uses character codes; binary stores values in internal formats.

  8. 8. If a hashing function gives the same address for two keys, this is called:

    Correct answer: B — collision.

  9. 9. Which encoding uses 7 bits per character (originally)?

    Correct answer: C — ASCII (7-bit).

  10. 10. Which statement about floating-point rounding errors is correct?

    Correct answer: B — rounding errors can accumulate.

  11. 11. Which file organisation is best for fast individual record lookup with many users?

    Correct answer: C — direct access.

  12. 12. Which of the following is NOT a property of a set data type?

    Correct answer: D — sets have no indexing or fixed positions.

  13. 13. What does 44.1 kHz refer to in audio?

    Correct answer: C — sampling rate (samples per second).

  14. 14. Which is the best description of an enumerated data type?

    Correct answer: B — enumerated type lists named values.

  15. 15. Which structure would you use to ensure 'no duplicate' student IDs?

    Correct answer: B — a set enforces uniqueness.

  16. 16. In fixed-point with 2 fractional bits, what is the smallest positive step?

    Correct answer: B — 1/4 (two fractional bits → 2^-2).

  17. 17. What is the main advantage of converting a list to a set when removing duplicates?

    Correct answer: B — sets eliminate duplicates by definition.

  18. 18. Which of the following is most likely to require a sequential file organisation?

    Correct answer: C — sequential suits batch processing and master files.

  19. 19. If you want maximum precision for fractional values, you should:

    Correct answer: B — more mantissa bits → greater precision.

  20. 20. Which audio format is LOSSY?

    Correct answer: C — MP3 is lossy.

  21. 21. Which is the MAIN reason for using binary files to store program data?

    Correct answer: B — binary files store values in internal representation.

  22. 22. Which is an advantage of indexed-sequential files?

    Correct answer: B — combines index speed with sequential order.

  23. 23. Which of these is a correct use of an enumerated type?

    Correct answer: B — enumerated values for compass directions.

  24. 24. When converting decimal fractions to binary, which fractional value converts exactly?

    Correct answer: C — 0.5 converts to 0.1₂ exactly.

  25. 25. Which of these is true about pointer variables?

    Correct answer: B — pointer variables store addresses in memory.
