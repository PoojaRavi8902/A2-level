Stacks in Python (Using LIST) – Cambridge AS & A Level 9618

Teacher Pane – Deep Explanation & Exam Coverage

1. What is a Stack?

A stack is a linear data structure that follows:

LIFO – Last In, First Out

The last item added is the first item removed.

Classic analogy: stack of plates.
2. Stack Terminology (Exam Keywords)
  • PUSH – add an item
  • POP – remove an item
  • PEEK / TOP – view top item
  • Stack Pointer (conceptual)
3. Why Use Python LIST for Stack?
  • Already known syntax
  • append() and pop() match PUSH and POP
  • Acceptable for Cambridge 9618
4. Creating an Empty Stack 
stack = []
5. PUSH Operation 
stack.append("A")
stack.append("B")
stack.append("C")

Stack becomes: ["A", "B", "C"]

6. POP Operation 
item = stack.pop()

Removes last element.

Stack becomes: ["A", "B"]

7. PEEK Operation 
top_item = stack[-1]
8. Stack Underflow 
if len(stack) == 0:
    print("Stack is empty")
Very common theory question.
9. Full Stack? (Theory)

In theory, stack has fixed size.

In Python, memory defines limit.

10. Complete Stack Program 
stack = []

stack.append(10)
stack.append(20)
stack.append(30)
print(stack)

removed = stack.pop()
print("Removed:", removed)
print(stack)
11. Stack Traversal (Exam) 
for item in stack:
    print(item)
12. Applications of Stack
  • Function calls
  • Undo / Redo
  • Expression evaluation
  • Backtracking

H. Challenge (A Level)

Write a menu-driven stack program.

Menu-Driven Stack Program (Cambridge 9618 – Exam Standard)

Complete Menu-Driven Stack Program 
# Stack using Python List
stack = []

while True:
    print("
STACK MENU")
    print("1. PUSH")
    print("2. POP")
    print("3. DISPLAY")
    print("4. EXIT")

    choice = int(input("Enter your choice: "))

    if choice == 1:
        item = input("Enter item to push: ")
        stack.append(item)
        print("Stack after PUSH:", stack)

    elif choice == 2:
        if len(stack) == 0:
            print("Stack is empty. POP not possible")
        else:
            removed = stack.pop()
            print("Popped item:", removed)
            print("Stack after POP:", stack)

    elif choice == 3:
        print("Current Stack:", stack)

    elif choice == 4:
        print("Exiting program")
        break

    else:
        print("Invalid choice")
